# Switch Ability

## ゲームの内容
丸、三角、四角それぞれの能力を駆使し、時と場合によって能力を変更しながら進んでいく2D謎解きアクションゲーム

![画面イメージ](docs/images/game_image01.png)

## 操作説明
ADで左右移動、Spaceでジャンプ、Qで能力変更
